# Pirates Bay Palooza Playtests

This is the public, binary-only download home for Pirates Bay Palooza playtest builds. The Unity source code is maintained separately and is not published here.

## Download

Use the [latest playtest release](https://github.com/atla/PiratesBayPalooza-Playtests/releases/latest). Each release contains:

- `PiratesBayPalooza-Windows-<version>.zip` - extract the complete ZIP and run `PiratesBayPalooza.exe`.
- `PiratesBayPalooza-Android-<version>.apk` - open the APK on an Android device and allow installation from that browser or file manager if prompted.
- `SHA256SUMS.txt` and `build-manifest.json` - integrity hashes and source/build provenance.

## Reporting feedback

Include the exact playtest version, platform, device or Windows version, and steps to reproduce the problem. Screenshots or short recordings are especially useful for visual and interaction issues.

These are development builds intended for testing and may contain unfinished features or known issues. See each release's changelog before playing.
